# KIN-0XX --- Mood Dot

Category: Mental Health\
Creator: Kin

## Concept

Mood Dot is an ultra-minimal emotional state tracker.

Instead of complicated journaling systems, the user simply selects one
of five mood dots representing their current emotional state. A short
optional note can be added for context.

The tool displays a rolling 7-day grid so users can visually detect
emotional patterns across the week.

Everything stays entirely local in the browser.

No accounts, no analytics, no tracking.

The goal is frictionless emotional check-ins that take less than 10
seconds.

------------------------------------------------------------------------

## Core Features

### Mood Dots

Five selectable mood states represented by colored dots:

1 --- Very Low\
2 --- Low\
3 --- Neutral\
4 --- Good\
5 --- Very Good

Clicking a dot logs the mood for the current day.

Large visual buttons optimized for mobile.

------------------------------------------------------------------------

### Optional Note

Small text field allowing a short note.

Examples:

-   Bad sleep
-   Great meeting
-   Stressful travel day

Notes are optional and kept intentionally short.

------------------------------------------------------------------------

### 7-Day Mood Grid

A visual grid showing the last seven days.

Each day displays:

-   mood dot color
-   abbreviated date
-   indicator if note exists

Helps reveal emotional patterns across a week.

Example layout:

Mon ●\
Tue ●\
Wed ●\
Thu ●\
Fri ●\
Sat ●\
Sun ●

------------------------------------------------------------------------

### Edit Today's Entry

User can:

-   change mood
-   edit note
-   delete note

Only the current day's entry is editable to keep the system simple.

------------------------------------------------------------------------

### Export Notes

Button: **Export Mood Notes**

Exports all notes and mood values as a plain text file.

Example export:

Date: 2026-03-15\
Mood: 4\
Note: Productive studio session

Date: 2026-03-16\
Mood: 2\
Note: Poor sleep

This allows users to keep a personal record outside the browser if
desired.

------------------------------------------------------------------------

## Data Storage

Local browser storage only.

No accounts.\
No cloud sync.\
No tracking.

Example data format:

    {
      "2026-03-15": {
        "mood": 4,
        "note": "Productive studio session"
      }
    }
