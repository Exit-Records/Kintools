# KIN-034 — Noise Meter

Single-file HTML classroom noise level meter. Uses Web Audio API to capture microphone input and display real-time noise levels as a green/yellow/red colour bar.

## Features
- Green (< 50 dB), yellow (50-75 dB), red (> 75 dB) zone metering
- Large status word (Quiet / Moderate / Too Loud) as primary visual
- Adjustable sensitivity slider for different mic hardware
- Screen Wake Lock API keeps display on during listening
- Re-acquires wake lock on tab visibility change
- PWA with service worker for offline use

## Tech
- Web Audio API (AnalyserNode, getFloatTimeDomainData)
- Screen Wake Lock API (graceful fallback)
- Single-file HTML, no dependencies, no build step
- Deployed to Netlify via Exit-Records/Kintools monorepo

## Files
- `index.html` — the tool (root, for Netlify deploy)
- `sites/kin-034/index.html` — same file (monorepo site path)
