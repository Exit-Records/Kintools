### KIN-015 Quiet Cycle (High risk — sensitive reproductive health data)

**Stores:** Cycle history, settings, PIN hash in `kin-quiet-cycle`

PASS:

- Single-file HTML, Blob manifest, Blob SW (kin015-v1)
- Most privacy-sensitive tool in the catalogue
- PIN lock with client-side hash (djb2 variant with salt, adequate for local-only)
- Panic delete via 5-tap on header (clears all data instantly)
- Cycle prediction: average of all intervals, applied to last start date
- Fertility window: ovulation at next_period - 14 days, window ov-2 to ov+1
- Correctly disclaimed as estimate only (“Not medical advice”)
- Calendar view with period days, predicted days, fertile days
- Cycle variability detection (warns if range > 7 days)
- Timeline with bar chart of cycle lengths
- Full CRUD on cycle entries with date/note fields
- Export reminder system with weekly/monthly frequency
- Data export as .txt
- Delete all data with confirm dialog
- Privacy guarantee panel with verifiable claims
- By Alice and Darren (correct credit)
- ⓘ storage popover present

ISSUES:

- [MEDIUM] XSS in renderEntries(): note field rendered via template literal in innerHTML
  Impact: Self-XSS if user enters HTML in notes
  Fix: Escape note text before insertion
- [LOW] No dark mode toggle (light only)
- [LOW] Viewport has `maximum-scale=1`
- [LOW] Bug report: old pattern
- [LOW] No OG/canonical tags
