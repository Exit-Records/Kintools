Kin

Everyday tools.
No strings.

Kin is a curated collection of simple, free web tools that work without subscriptions, accounts, or tracking. A tuner. A timer. A colour picker. A drum machine. Built by real people, credited to them, and designed to last.

The problem Kin is solving is not a technical one. These tools are trivially simple to build. What happened is that a business model captured them. The subscription economy turned every small utility into a recurring revenue opportunity. Most people assume that if something is useful it must cost money — or harvest their data instead. That assumption is wrong.

Kin is structured like a record label. Each tool is a release: catalog number, creator credit, consistent visual identity. When you see the Kin mark, you know — this is free, this is clean, this works, someone checked.

The technical bar to contributing is deliberately low. A single HTML file can be a complete Kin tool. The harder bar is quality and intent: does it genuinely help someone do something without asking for anything in return?

If this resonates — whether you want to build something, use something, or just talk about it — I'd like to hear from you.
